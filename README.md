Proof of Concept. It's not production-ready.

GraphQL API scaffolding made simple.

## Prerequisites

- Node.js
- MongoDB
- Yarn (optional)

## Installation

```
$ yarn
$ yarn dev # development mode
$ yarn start # server mode
```

## Usage

Modify the `shards.config.js` file and add your models with their props.
